— Problem Analysis
기존 OTT의 UX 문제
01
선택 피로 (Choice Overload)
추천 콘텐츠가 너무 많아 오히려 선택하지 못하고 스크롤만 반복하는 현상 발생
02
직관적이지 않은 장르 필터
장르 분류 기준이 모호하고 사용자 취향 기반 탐색보다 카테고리 탐색 중심 구조
03
긴 콘텐츠 선택 시간
콘텐츠 선택까지 평균 3~5분 소요. 모바일에서 탐색 동선이 불필요하게 긴 문제

— Key UX Features
핵심 UX 개선안
01
Mood 기반 콘텐츠 탐색
오늘의 기분(가볍게·긴장감·로맨스·영화 등)을 선택하면 해당 분위기의 콘텐츠를 즉시 추천. 장르 카테고리보다 직관적.
Mood Filter
02
30초 콘텐츠 선택 UX
3개 질문(오늘 기분은? / 시청 시간은? / 장르는?)에 답하면 즉시 추천. 기존 3~5분 탐색을 30초로 단축.
Quick Pick
03
스크롤 탐색 최소화
선택 기반 탐색 구조로 무한 스크롤을 줄이고 원하는 콘텐츠에 빠르게 도달할 수 있는 네비게이션 설계.
Focused Nav
04
경쟁 서비스 분석 기반
Netflix·Watcha·Disney+ 탐색 구조·추천 방식을 Jakob Nielsen UX 원칙 적용하여 비교 분석 후 개선안 도출.

"Netflix·Watcha·Disney+의 콘텐츠 탐색 UX를 비교 분석해줘. 사용자가 콘텐츠를 발견하는 방식, 필터 기능, 추천 알고리즘 노출 방식 중심으로."
→ 3개 서비스 탐색 구조 비교표, 공통 문제점 도출, UX 개선 방향 3가지 제안

Mood 기반 추천 UX 설계
"사용자 기분 기반으로 콘텐츠를 추천하는 UI 플로우를 설계해줘. 3개 이하의 선택으로 즉시 추천이 나오는 구조. 모바일 퍼스트."
→ Mood → Genre → Time 3단계 Quick Pick 플로우, UI 컴포넌트 구조 완성
