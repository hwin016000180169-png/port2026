CLASS
CARE

학원 및 학교 환경에서 학생·교사 간 수업 관리와 커뮤니케이션을 효율적으로 지원하는 교육용 플랫폼. 파일 공유·모의 시험·Gemini AI 오답 노트를 하나의 시스템으로 통합. 실제 학원에서 운영 중.

Live Demo ↗ GitHub

— Key Features
핵심 기능
01
파일 전송 시스템
교사→학생 전체 자료 배포, 학생→교사 과제 제출. 1:다수 / 다수:1 방식 지원
02
실시간 커뮤니케이션
수업 중 질문·메모·공지를 플랫폼 내에서 실시간 전달. 교사-학생 직접 소통
03
모의 시험 시스템
시험 배포 → 학생 응답 제출 → 결과 자동 분석. 통합 관리로 시험 효율화
04
AI 오답 노트 (Gemini)
Gemini API로 시험 결과 분석. 틀린 문제 유형 분류 → 자동 오답 노트 생성

— AI Collaboration
AI 프롬프트 예시
Gemini API
AI 오답 노트 기능
$ prompt
"학생의 시험 결과 JSON 데이터를 분석해서 틀린 문제 유형을 분류하고, 각 문제에 대한 오답 이유와 보완 학습 방향을 자동으로 생성해줘."
→ output
문제 유형별 분류, 오답 원인 분석, 반복 학습 포인트 자동 생성. 실제 Gemini API 연동 구현
Claude
UI 프로토타입 제작
$ prompt
"교사와 학생이 각각 다른 권한으로 접속하는 대시보드 UI를 만들어줘. 역할 기반 뷰 분리, 교육 앱 스타일."
→ output
역할 기반 대시보드 레이아웃, 사이드바 네비게이션, 컴포넌트 구조 설계 완성
