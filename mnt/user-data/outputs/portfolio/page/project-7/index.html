01
노후화된 관리자 UI

02
외부 플랫폼 종속 구조
메인·서브 페이지 대부분이 nncplus.com 외부 링크에 의존, 자체 콘텐츠 관리가 불가능한 구조
03
서비스 신청 관리 부재
식판 렌탈 서비스 신청 접수·승인·계정 발급 흐름이 수동 처리되어 운영 비효율 발생

— Core Work
MIGRATION & REBUILD
01
외부 링크 전면 제거
전 페이지에 산재된 nncplus.com onclick 링크를 제거하고 자체 내부 라우팅으로 대체. 외부 의존도 0%로 전환.
02
공통 레이아웃 컴포넌트화
GNB·사이드바·푸터를 layout.js 단일 소스로 통합 관리. 80+ 페이지에 일괄 적용 가능한 구조 설계.
03
관리자 UI 전면 리디자인
구형 화이트 테마에서 다크 네이비 테마로 전환. NNC 브랜드 컬러(#1A6BFF) 기반 디자인 시스템 구축.
04
메인·서브페이지 리모델링
홈페이지·회사소개·CS·서비스 페이지를 신규 설계. 기존 외부 플랫폼의 콘텐츠를 자체 퍼블리싱으로 이전 중.

FEATURE IMPROVEMENTS
📋 NEW
서비스 신청 관리
식판 렌탈 서비스 신청 접수 → 승인 → 계정 발급까지 원스톱 처리. 상태별 탭 필터 및 엑셀 저장 지원.
⚡ IMPROVED
대시보드 KPI & 빠른 실행
오늘 주문·신규 가입·대기 신청·이번 달 매출을 한눈에. 자주 쓰는 메뉴 빠른 실행 단축 카드 추가.
📊 IMPROVED
통계분석 고도화
접속자·주문·가입 통계를 시간·일·주·월·연도별 다차원 분석. 기간 직접 입력 필터 및 엑셀 저장 기능.
🏪 IMPROVED
가맹점 관리 체계화
신규신청·연장·수수료정책·정산·트리구조를 독립 모듈로 분리. 가맹점 주문통계 및 추천인 변경 로그 추가.
👥 IMPROVED
회원 관리 기능 확장
레벨 필터·구매수·로그인 이력·접근차단 컬럼 추가. 전체 메일 발송·문자 발송·엑셀 일괄 처리 지원.
🎨 IMPROVED
디자인 관리 모듈
배너·팝업·로고·모바일 배너·스킨·콘텐츠를 관리자 화면에서 직접 편집·관리할 수 있는 UI 구성.
📄 NEW
CMS 동의서 PDF 다운로드
선택 건 또는 전체 동의서를 A4 1페이지 PDF로 클라이언트 사이드 변환 저장. html2canvas + jsPDF 활용, 서버 부하 없이 브라우저에서 즉시 생성.
🏫 NEW
원장 전용 대시보드
어린이집·유치원 원장이 쇼핑몰 로그인 시 '내 원 현황' 버튼 자동 노출. 클릭 시 본인 원의 반별 아이 목록·인원·신청일을 권한 분리 없이 회원 ID 기반으로 표시.
🏷️ NEW
원명 자동완성 칩 UI
그룹 설정 시 담당 원명을 수동 입력 없이 기존 신청 DB 기반 칩으로 원클릭 입력. 신청 건수와 함께 자동 표시되며 datalist와 동기화.
